Lesson 1: Using Your Web Browser

Before we start navigating the web, letâ€™s learn some basics about using your web browser. The commands listed below will help you navigate the browser's menus, the address bar, and will help you manage open tabs.

You may recognize some of these commands from the SR101 - Basic Computer Navigation course. You'll still use Alt to reach the program menu. You'll still use Alt + left/right arrows to go back or forward between visited pages. As you work through this course, think about how these new commands relate to the commands you're already using.

Task Keyboard Command
Move to Browser Menu Alt
Move to Address Bar Ctrl + L or Alt + D
Go Back to Previous Web Page Alt + Left Arrow
Go Forward to (Previous) Web Page Alt + Right Arrow
Switch to next browser tab Ctrl + Tab
Switch to previous browser tab Shift + Ctrl + Tab
Open a new tab Ctrl + T
Close a tab Ctrl + W
Re-open closed tab Ctrl + Shift + T
Show/Hide Bookmarks Bar Ctrl + Shift + B
